Statically Indeterminate Beams

Beams classified as statically indeterminate present a challenge in determining reactions through conventional rigid body equilibrium analysis. This complexity arises due to the imbalance between the number of unknowns and the available equilibrium equations. In such scenarios, the usual methods of solving for reactions become insufficient.

This is a simple web application that calculates the reactions and internal forces of statically indeterminate beams. The application uses the slope-deflection method to solve the beam. The user is required to input the necessary information about the beam, and the application will calculate the reactions and internal forces.
